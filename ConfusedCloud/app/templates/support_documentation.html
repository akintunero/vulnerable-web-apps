Getting Started

Welcome to ConfusedCloud! This guide will help you get started with our cloud platform.

Quick Setup

Install ConfusedCloud CLI 
curl -sSL https://get.confusedcloud.com/cli | bash
cc configure --api-key YOUR_API_KEY

Your First Instance

Create a compute instance 
cc compute create \
  --name my-first-instance \
  --type t2.micro \
  --image ubuntu-20.04 \
  --region us-east-1

API Reference

Our REST API provides programmatic access to all ConfusedCloud services.

Base URL

https://api.confusedcloud.com/v1

Authentication

All API requests require authentication using your API key in the Authorization header. 

Authorization: Bearer YOUR_API_KEY

Authentication

ConfusedCloud uses API key-based authentication for secure access to your resources.

Generating API Keys

  1. Navigate to your Dashboard
  2. Go to API Keys section
  3. Click "Generate New Key"
  4. Copy and securely store your key
Security Note: Keep your API keys secure and never commit them to version control.

Storage Services

Scalable object storage, block storage, and file systems for your applications.

Object Storage

# Upload a file
cc storage upload my-bucket my-file.txt

# Download a file
cc storage download my-bucket my-file.txt

# List objects
cc storage list my-bucket

Database Services

Managed database services including MySQL, PostgreSQL, and MongoDB.

Creating a Database

cc database create \
  --name my-database \
  --engine mysql \
  --version 8.0 \
  --size 20

Networking

Virtual networks, load balancers, and DNS management.

Load Balancer

cc loadbalancer create \

Code Examples

Practical examples for common use cases.

Python Example

import confusedcloud

# Initialize client
client = confusedcloud.Client(api_key='your-api-key')

# Create an instance
instance = client.compute.create_instance(
    name='web-server',
    type='t2.micro',
    image='ubuntu-20.04'
)

print(f"Instance created: {instance.id}")

Node.js Example

const { ConfusedCloud } = require('@confusedcloud/sdk');

const client = new ConfusedCloud({
  apiKey: 'your-api-key'
});

// Create storage bucket
const bucket = await client.storage.createBucket({
  name: 'my-bucket',
  region: 'us-east-1'
});

console.log(`Bucket created: ${bucket.name}`);
