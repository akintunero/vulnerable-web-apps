SkyHack Airlines API Documentation

Security Notice: All API access is monitored by QuantumShield AI and SkyHack SIEM. Unauthorized use is strictly prohibited.
Base URL
https://api.skyhack-airlines.com/v1/
Authentication

Use your API key in the X-API-Key header.

Example API Key: sk_live_12345FAKEKEY67890
Endpoints
  • GET /flights - List all flights
  • POST /book - Book a flight
  • POST /checkin - Check in for a flight
  • GET /loyalty - Get loyalty status
  • POST /transfer-points - Transfer loyalty points
  • GET /siem/events - View SIEM events (admin only)
  • GET /admin/backup - Download admin backup (403 Forbidden)
  • GET /api/internal - Internal API (access restricted)
  • POST /api/flag - Honeypot endpoint (returns error and logs access)
  • GET /staff/portal - Staff portal (access restricted)
  • GET /api/hidden-status - Returns hidden status (misleading)
  • POST /api/broken-endpoint - (Deprecated) Always returns 500 error
  • GET /api/experimental/quantum - (Experimental) May return random data
Deprecated/Experimental Endpoints
  • POST /api/broken-endpoint - Deprecated, unstable, always fails
  • GET /api/experimental/quantum - Experimental, may be removed at any time
Security Warnings
  • All API requests are logged and analyzed by QuantumShield AI.
  • Multiple failed authentication attempts will result in a temporary ban.
  • API keys are rotated every 24 hours for your security.
  • Do not share your API key. Violations will be reported to SkyHack SOC.
